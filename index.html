FragmentQR

Yonah Aviv

FragmentQR is a web extension that generates QR codes linking to arbituary selected text on a webpage, which will highlighted and scrolled to when opened in a compatible browser.

Tutorial

  1. Select text
  2. Ctrl/Cmd + Shift + L to copy the text fragment. Wait for your selected text be highlighted in yellow.
  3. Ctrl/Cmd+Q to open the extension popup
  4. Click the paste button
  5. Scan it on your mobile device using a QR code scanner such as Google Lens or some default camera apps.
And thats it! If steps 1 or 2 do not work, go to chrome://extensions/shortcuts to reconfigure the shortcuts

Compatible Browsers:

Mac and Windows: Chromium 80+.

Android: Chrome 80+ and some Chromium 80+ browsers

iOS: Latest version of Google Chrome if #shared-highlighting-ios enabled on chrome://flags.

Features

  • Generates a QR code from the url of current tab when clicking icon
  • Create link to text fragment that highlights and scrolls to any text when opened
  • Copy, paste and clear buttons
  • Editable text-field
  • Light and dark theme
  • Enlargable QR code
  • Download QR code as png
  • Clean UI with tooltips and Material icons
